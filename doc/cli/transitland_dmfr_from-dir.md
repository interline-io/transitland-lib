## transitland dmfr from-dir

Generate DMFR from a directory of GTFS files

### Synopsis

Generate DMFR from a directory of GTFS files

Scans a directory for .zip files and outputs a DMFR registry JSON to stdout.

```
transitland dmfr from-dir [flags] <directory>
```

### Options

```
  -h, --help            help for from-dir
      --prefix string   Prefix for generated feed IDs
      --recursive       Search subdirectories recursively
```
